# 🚀 Amazon SDE II Interview Prep Tool

An advanced AI-powered interview preparation tool similar to LockedIn AI, specifically designed for Amazon SDE II interviews.

## ✨ Features

- **🤖 AI Chat Coach**: Get personalized guidance and feedback
- **📝 Mock Interviews**: Practice DSA, System Design, and Behavioral questions
- **⭐ STAR Format Training**: Structured behavioral interview practice
- **📊 Progress Tracking**: Detailed analytics and performance trends
- **🎯 Readiness Score**: Track your overall interview preparation
- **📚 Curated Resources**: Links to essential preparation materials

## 🛠️ Technology Stack

- **Frontend**: Streamlit
- **AI**: Claude API (Anthropic)
- **Visualization**: Plotly
- **Data Processing**: Pandas

## 🚀 Quick Start

### Option 1: Deploy to Streamlit Cloud (Recommended)

1. **Fork this repository** to your GitHub account

2. **Get Claude API Key**:
   - Go to [Anthropic Console](https://console.anthropic.com/)
   - Create an account and get your API key

3. **Deploy to Streamlit Cloud**:
   - Go to [Streamlit Cloud](https://share.streamlit.io/)
   - Connect your GitHub account
   - Select your forked repository
   - Set main file path: `main_app.py`
   - Click Deploy!

4. **Configure API Key**:
   - Once deployed, enter your Claude API key in the sidebar
   - Start practicing!

### Option 2: Run Locally

1. **Clone the repository**:
```bash
git clone <your-repo-url>
cd amazon-sde-interview-prep
```

2. **Install dependencies**:
```bash
pip install -r requirements.txt
```

3. **Run the application**:
```bash
streamlit run main_app.py
```

4. **Open in browser**:
   - The app will open at `http://localhost:8501`
   - Enter your Claude API key in the sidebar

## 📋 Interview Preparation Guide

### 3-Day Intensive Plan

#### Day 1: Foundation & DSA
- ✅ Complete 8-10 DSA problems (Arrays, Strings, Hash Maps)
- ✅ Practice 2 behavioral questions with STAR format
- ✅ Review Amazon Leadership Principles

#### Day 2: System Design & Advanced DSA
- ✅ Complete 2 system design mock interviews
- ✅ Solve 6-8 advanced DSA problems (Trees, Graphs, DP)
- ✅ Practice 3 more behavioral scenarios

#### Day 3: Mock Interviews & Review
- ✅ Full mock interview session (all rounds)
- ✅ Review weak areas identified by the AI coach
- ✅ Final preparation and confidence building
